â“ Masalah Umum dan Solusi

ğŸš« Aplikasi Tidak Bisa Dibuka

Gejala:

Double-click Pendita.exe tidak menjalankan aplikasi atau muncul error message.

Solusi:

  1. Pastikan menjalankan sebagai Administrator
  2. Check apakah semua file aplikasi lengkap (tidak corrupt)
  3. Disable temporary antivirus yang mungkin blocking
  4. Re-extract file ZIP jika diperlukan
  5. Check system compatibility (Windows 10+)

ğŸ“ File Tidak Bisa Dibuka

Gejala:

File dialog muncul error atau file terpilih tidak bisa ditampilkan.

Solusi:

  1. Pastikan file tidak corrupt atau terkunci
  2. Check format file didukung oleh Pendita
  3. Pastikan file tidak sedang digunakan aplikasi lain
  4. Try pindahkan file ke lokasi lain
  5. Check permission file (tidak read-only)

ğŸ“… Kalender Tidak Muncul

Gejala:

Area kalender kosong atau tidak menampilkan tanggal.

Solusi:

  1. Klik tombol "ğŸ”„ Refresh Kalender"
  2. Navigate ke halaman lain dan kembali ke Home
  3. Refresh browser dengan F5 (jika diperlukan)
  4. Restart aplikasi Pendita
  5. Check system date/time settings

ğŸ–Šï¸ Papan Tulis Lag atau Tidak Responsif

Gejala:

Drawing tools lambat merespon atau hasil gambar tidak smooth.

Solusi:

  1. Reduce brush size jika terlalu besar
  2. Clear browser cache jika aplikasi web-based
  3. Close aplikasi lain yang menggunakan RAM berlebih
  4. Check hardware acceleration settings
  5. Update graphics drivers

ğŸ’¾ File Tidak Bisa Disimpan

Gejala:

Save dialog error atau file tidak tersimpan di lokasi yang dipilih.

Solusi:

  1. Check permission folder tujuan (not read-only)
  2. Pastikan ruang storage mencukupi
  3. Gunakan path yang lebih pendek
  4. Avoid special characters di nama file
  5. Try save ke location lain (Desktop/Documents)

ğŸ†˜ Bantuan Lebih Lanjut

Jika masalah persisten:

  1. Screenshot error message yang muncul
  2. Note down langkah-langkah yang menyebabkan error
  3. Check Windows Event Viewer untuk error details
  4. Hubungi developer atau support team
  5. Consider reinstall aplikasi sebagai last resort
