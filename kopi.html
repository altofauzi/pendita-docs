Panduan Pengguna Kopi

Selamat datang di panduan pengguna aplikasi Kopi (Koperasi Simpan Pinjam). Panduan ini akan membantu Anda memahami cara menggunakan seluruh fitur yang tersedia.

Apa itu Kopi?

Kopi adalah aplikasi desktop untuk manajemen koperasi simpan pinjam. Dibangun menggunakan teknologi Electron + React, mendukung pengelolaan anggota, simpanan, pinjaman, pembayaran cicilan, dan pelaporan.

Fitur Utama

  • Manajemen data anggota koperasi
  • Pencatatan simpanan (Pokok, Wajib, Sukarela)
  • Pengelolaan pinjaman dan jadwal cicilan otomatis
  • Pembayaran pinjaman dengan tracking status
  • Buku tabungan digital per anggota
  • Cetak laporan dalam format PDF
  • Dark mode
  • Sistem lisensi dengan aktivasi product key

Langkah Instalasi

  1. Download file installer Kopi dari sumber yang diberikan.
  2. Jalankan file installer (Kopi-Setup.exe).
  3. Ikuti langkah instalasi hingga selesai.
  4. Buka aplikasi Kopi dari desktop atau Start Menu.
  5. Aplikasi akan otomatis dalam mode Trial 30 hari dengan fitur lengkap.

Form Simpanan

Akses melalui: Menu Simpanan > Form Simpanan atau tekan Ctrl + Alt + E

Langkah Menambah Simpanan

  1. Pilih Anggota dari daftar.
  2. Pilih Buku yang akan digunakan.
  3. Pilih Jenis Simpanan (Pokok / Wajib / Sukarela).
  4. Pilih Tipe (Masuk / Keluar).
  5. Masukkan Jumlah nominal.
  6. Isi Keterangan jika perlu.
  7. Klik Simpan.
Info: Setiap transaksi simpanan otomatis membuat record mutasi tabungan dan memperbarui saldo buku.

Detail Pinjaman

Klik pada baris pinjaman untuk melihat detail yang mencakup:

  • Informasi pinjaman (nomor, jumlah, tenor, bunga)
  • Jadwal cicilan per bulan
  • Status pembayaran setiap cicilan
  • Tanggal jatuh tempo

Form Pinjaman

Akses melalui: Menu Pinjaman > Form Pinjaman atau tekan Ctrl + Alt + G

Langkah Membuat Pinjaman Baru

  1. Pilih Anggota peminjam.
  2. Masukkan Jumlah Pinjaman (pokok pinjaman).
  3. Masukkan Tenor (lama cicilan dalam bulan).
  4. Masukkan Bunga (persentase bunga).
  5. Klik Simpan.
Info: Sistem otomatis membuat jadwal cicilan per bulan. Setiap cicilan terdiri dari pokok + bunga. Tanggal jatuh tempo dihitung ke akhir bulan.

Rumus Perhitungan

KomponenRumus
Cicilan PokokJumlah Pinjaman ÷ Tenor
Cicilan Bunga(Jumlah Pinjaman × Bunga%) ÷ Tenor
Total per BulanCicilan Pokok + Cicilan Bunga

Bayar Pinjaman

Akses melalui: Menu Pinjaman > Bayar Pinjaman atau tekan Ctrl + Alt + H

Langkah Pembayaran

  1. Buka Form Bayar Pinjaman (Ctrl + Alt + J).
  2. Pilih Pinjaman yang akan dibayar.
  3. Pilih Cicilan yang akan dilunasi (bisa lebih dari satu).
  4. Pilih Metode Pembayaran (Cash / Transfer / E-Wallet).
  5. Isi data Cashier dan Dibayar Oleh.
  6. Klik Proses Pembayaran.
Tips: Nomor slip pembayaran otomatis di-generate oleh sistem dengan format BP-YYYYMMDD-XXXX.
Info: Jika semua cicilan sudah lunas, status pinjaman otomatis berubah menjadi Lunas.

Dark Mode

Kopi mendukung tampilan dark mode untuk kenyamanan penggunaan di lingkungan minim cahaya.

Cara Mengaktifkan

Klik ikon bulan/matahari di pojok kanan atas header.

  • Ikon bulan = mode terang aktif (klik untuk pindah ke dark)
  • Ikon matahari = mode gelap aktif (klik untuk pindah ke terang)
Tips: Preferensi tema tersimpan otomatis dan akan digunakan kembali saat aplikasi dibuka.

Masa Trial

Saat pertama kali diinstall, aplikasi berada dalam mode Trial 30 hari.

Selama Masa Trial

  • Semua fitur tersedia tanpa batasan
  • Banner trial muncul di bagian atas aplikasi
  • Banner berwarna kuning jika sisa > 7 hari
  • Banner berwarna merah jika sisa ≤ 7 hari
  • Banner dapat ditutup (dismiss) oleh pengguna

Setelah Trial Habis

  • Aplikasi terblokir — tidak bisa mengakses fitur apapun
  • Halaman Aktivasi Lisensi tampil otomatis
  • Pengguna harus melakukan aktivasi product key untuk melanjutkan

Aktivasi Lisensi

Aktivasi lisensi diperlukan setelah masa trial berakhir, atau dapat dilakukan kapan saja melalui menu Pengaturan > Aktivasi Lisensi.

Langkah Aktivasi

  1. Salin Hardware ID
    Buka halaman Aktivasi. Hardware ID unik perangkat Anda akan ditampilkan. Klik tombol Salin untuk menyalin ke clipboard.
  2. Hubungi Penjual
    Kirimkan Hardware ID + email aktif Anda ke penjual/developer melalui WhatsApp, email, atau media lainnya. Sertakan bukti pembayaran jika diperlukan.
  3. Terima Product Key
    Penjual akan mengirimkan product key dengan format: KOPI-XXXX-XXXX-XXXX-XXXX
  4. Masukkan Product Key
    Di halaman Aktivasi, masukkan product key dan email Anda. Key akan di-format otomatis (huruf besar + dash).
  5. Klik Aktivasi
    Sistem akan memvalidasi ke server. Jika berhasil, aplikasi langsung bisa digunakan.
Perhatian: Aktivasi memerlukan koneksi internet. Pastikan perangkat terhubung ke internet saat proses aktivasi.

FAQ (Pertanyaan Umum)

Bagaimana cara menambah user baru?

Saat ini user hanya bisa ditambahkan melalui database. Hubungi administrator untuk penambahan user baru.

Bagaimana jika lupa password?

Hubungi administrator untuk mereset password melalui database.

Apakah data tersimpan di cloud?

Tidak. Semua data tersimpan secara lokal di komputer Anda dalam file SQLite. Pastikan melakukan backup secara berkala.

Bagaimana cara backup data?

Salin file kopi.sqlite dari folder aplikasi ke lokasi backup Anda. File ini berisi seluruh data koperasi.

Bagaimana jika ingin pindah komputer?

1. Salin file kopi.sqlite ke komputer baru.
2. Install aplikasi Kopi di komputer baru.
3. Hubungi penjual untuk memindahkan lisensi ke Hardware ID baru.
4. Aktivasi ulang dengan product key yang sama.

Apakah perlu koneksi internet?

Koneksi internet hanya diperlukan saat:
• Aktivasi lisensi (pertama kali)
• Re-validasi lisensi (setiap 30 hari, otomatis)

Untuk penggunaan sehari-hari, aplikasi dapat digunakan secara offline.
